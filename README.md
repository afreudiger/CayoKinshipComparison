This repository includes all code that was used for the manuscript Freudiger et al. "Genetic estimates of relatedness: Established practices and new opportunities through low coverage whole genome sequencing".
It includes the following files:
1. Analysis_LiteratureReview.R: Code for statistical analysis and plotting of literature review data.
2. Calculate_STRRelatedness.R: Code to calculate STR-based relatedness using the package RELATED.
3. Extract_ExomPositions.txt: Code to retrieve exom positions in the rhesus macaque genome and subsample to create subsets of the total exom.
4. Calculate_SNPRelatedness.txt: Code to calculate SNP-based relatedness using PLINK and ANGSD.
5. Calculate_PMRRelatedness.txt: Code to calculate PMR-based relatedness.
6. Analysis_ComparisonRelatednessEstimators.R: Code to compare the performance of STR-, SNP-, and PMR-based relatedness.
