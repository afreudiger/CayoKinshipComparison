This repository includes all code that was used for the manuscript Freudiger et al. "Genetic estimates of relatedness: Established practices and new opportunities through low coverage whole genome sequencing".
It includes the following files:
1. LiteratureReview01_FullTextAnalysis.R: Code for statistical analysis and plotting of literature review data.
2. RelatAnalysis01_STRRelatedness.R: Code to calculate STR-based relatedness using the package RELATED.
3. RelatAnalysis02_ExtractExomPositions.txt: Code to retrieve exom positions in the rhesus macaque genome and subsample to create subsets of the total exom.
4. RelatAnalysis03_SNPRelatedness.txt: Code to calculate SNP-based relatedness using PLINK and ANGSD.
5. RelatAnalysis04_ComparisonRelatednessEstimators.R: Code to compare the performance of STR-, SNP-, and PMR-based relatedness.
   <i>Note: Code to calculate PMR-based relatedness is available here: https://github.com/hringbauer/cayo_pmr/</i>
